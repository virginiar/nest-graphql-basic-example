# Nest-Basic-Example

API básico para gestión de una lista de TODO creado con [Nest](https://github.com/nestjs/nest). Basado en el curso de "Nest + GraphQL: Evoluciona tus APIs" de [DevTalles](https://cursos.devtalles.com/) en Udemy.

## Configuración del proyecto

```bash
$ npm install
```

## Compilar y ejecutar el proyecto

```bash
# desarrollo
$ npm run start

# modo observación
$ npm run start:dev

# modo producción
$ npm run build
$ npm run start:prod
```
